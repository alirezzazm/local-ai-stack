# 🚀 راهنمای اجرا (QUICKSTART)

این راهنما کامل می‌گوید چطور DAZ را روی سیستم خودت بالا بیاوری و تست کنی.

---

## ۰) پیش‌نیازها (یک‌بار نصب)
| ابزار | چرا | لینک |
|------|-----|------|
| **Git** | برای گرفتن پروژه | git-scm.com |
| **Node.js (نسخه ۱۸ یا بالاتر)** | برای رابط و سرور | nodejs.org |
| **Ollama** | موتور اجرای مدل | لازم نیست دستی نصب کنی؛ اسکریپت `setup` خودش نصب می‌کند |

> ویندوز: Git و Node را نصب کن. لینوکس/مک: معمولاً از قبل هستند یا با پکیج‌منیجر نصب می‌شوند.

---

## ۱) گرفتن پروژه
```bash
git clone https://github.com/alirezzazm/local-ai-stack
cd local-ai-stack
```

## ۲) نصب خودکار (مدل + Ollama)
**ویندوز (PowerShell):**
```powershell
./scripts/setup.ps1 -Profile auto      # سخت‌افزار را می‌خواند و مدل مناسب را دانلود می‌کند
ollama pull qwen2.5:7b                  # مدل پایه‌ی DAZ (روی سیستم ضعیف: qwen2.5:3b)
./scripts/build-fa.ps1                  # ساخت مدل فارسی DAZ
```
**لینوکس/مک (bash):**
```bash
chmod +x scripts/*.sh
./scripts/setup.sh auto
ollama pull qwen2.5:7b
./scripts/build-fa.sh
```

## ۳) اجرای رابط
```powershell
./scripts/webui.ps1            # لینوکس/مک: ./scripts/webui.sh
```
بعد در مرورگر باز کن: **http://localhost:8080**

---

## ۴) اولین استفاده
1. **شخصیت:** دکمه‌ی 👤 شخصیت → اسمت و لحن دلخواه را بگذار → ذخیره.
2. **چت:** پایین بنویس و Enter بزن. (تیک «⚙️ اقدامات» را فقط وقتی بزن که می‌خواهی کاری انجام دهد.)
3. **یادگیری:** دکمه‌ی 🌐 یادگیری → یک موضوع/لینک/فایل بده تا یاد بگیرد.
4. **مأموریت:** دکمه‌ی 🧠 مأموریت → یک هدف بده تا خودش تصمیم بگیرد و انجام دهد.

---

## ۵) دسترسی از گوشی / بیرون (اختیاری)
```powershell
$env:DAZ_TOKEN="یک‌رمز‌قوی"; $env:HOST="0.0.0.0"; ./scripts/webui.ps1
```
- همان شبکه: `http://IP-سیستمت:8080` (و رمز را وارد کن)
- از اینترنت بدون IP ثابت: `cloudflared tunnel --url http://localhost:8080`
- نصب به‌عنوان اپ: در مرورگر → منو → Install / Add to Home Screen

---

## ۶) سرور قوی با GPU (سریع و چند کاربر)
```bash
./scripts/setup.sh auto                                  # خودش server/strong را می‌گیرد
./scripts/serve-vllm.sh Qwen/Qwen2.5-32B-Instruct        # موتور پرسرعت
DAZ_BACKEND=openai OPENAI_BASE=http://localhost:8000/v1 OPENAI_MODEL=daz DAZ_TOKEN=رمز ./scripts/webui.sh
```

## ۷) رفع اشکال (Troubleshooting)
| مشکل | علت / راه‌حل |
|------|--------------|
| **جواب خیلی دیر می‌آید** | روی CPU طبیعی است (مدل کند است). تیک «اقدامات» را خاموش کن تا چتِ استریم سریع‌تر باشد. روی GPU بسیار سریع‌تر است. |
| **فارسی ضعیف است** | مدل پایه را Qwen بگذار (نه Llama). روی GPU مدل بزرگ‌تر = فارسی بهتر. |
| **صفحه باز نمی‌شود از بیرون** | `HOST=0.0.0.0` بگذار، پورت را در فایروال باز کن، یا از تونل cloudflared استفاده کن. |
| **میکروفون/عکس کار نمی‌کند** | روی `localhost` یا HTTPS کار می‌کند؛ برای موبایل از تونل HTTPS استفاده کن. |
| **عکس را نمی‌فهمد** | `ollama pull llava` و در UI مدل بینایی را llava بگذار. |
| **می‌خواهی DAZ کارها را اجرا کند (خاموش‌کردن/دستور)** | `DAZ_ALLOW_SYSTEM=1` را قبل از اجرا تنظیم کن (با احتیاط). |

## ۸) دستورهای پرکاربرد (Cheat Sheet)
```bash
ollama list                 # مدل‌های نصب‌شده
ollama run daz              # چت در ترمینال
./scripts/detect.ps1        # تشخیص سخت‌افزار و پروفایل پیشنهادی
./scripts/build-fa.ps1      # بازساخت DAZ بعد از تغییر شخصیت/مدل
```

## ۹) متغیرهای محیطی مهم
| متغیر | کار |
|------|-----|
| `PORT` | پورت رابط (پیش‌فرض ۸۰۸۰) |
| `HOST` | `0.0.0.0` برای دسترسی از بیرون |
| `DAZ_TOKEN` | رمز دسترسی (هنگام افشا حتماً بگذار) |
| `DAZ_BACKEND` | `ollama` یا `openai` (vLLM) |
| `DAZ_SUMMARY_MODEL` | مدل سبک برای یادگیری/خلاصه (پیش‌فرض qwen2.5:3b) |
| `DAZ_ALLOW_SYSTEM` | `1` تا اقدامات خطرناک (خاموش/دستور) فعال شود |
