**خلاصه فارسی از متن‌ها درباره DotNet**

.NET، کتابخانه‌ای برای توسعه برنامه‌های رایانه‌ای است که به زبان C# همگام با چند سیستم عامل فعال استفاده می‌شود. این کتابخانه پشتیبانی از سیستم‌های Linux، macOS و Windows دارد. .NET در حال حاضر به صورت ترقی‌نشده بر روی نسخه‌های 10.0 هست و نرم‌افزار SDK (Software Development Kit) مورد نیاز برای خواندن از آن تنظیم شده است. نسخه 10.0 با دامنه زمان پشتیبانی طولانی‌تر تازه‌کرده و کاربرد هماهنگ به فراهم کرده است.

**به روز بودن و دقیقی:**
- .NET نسخه 10.0 با دامنه زمان پشتیبانی طولانی‌تر (محدود به 5 سال) در حال خریداری می‌شود.
- برای خواندن از .NET بر روی Windows، macOS و Linux تنظیمات نرم‌افزار SDK
